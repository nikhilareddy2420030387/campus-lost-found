# Campus Lost and Found Management System

## 📌 Project Description

The Campus Lost and Found Management System is a full-stack web application designed to help students and staff report, search, and claim lost items within a campus. The system improves the process of recovering lost belongings by providing a centralized platform.

---

## 🚀 Features

* User login (Student / Staff)
* Report lost and found items
* View all items
* Claim items
* Mark items as recovered (Admin)
* Search and filter items
* Image upload for proof

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Storage:** In-memory data (can be extended to database)
* **Deployment:** Render

---

## 🔁 How It Works

1. User logs in using ID and password
2. Frontend sends request to backend using Fetch API
3. Backend processes request using Express APIs
4. Data is returned in JSON format
5. UI updates dynamically

---

## 📂 Project Structure

* `index.html` → Main UI
* `style.css` → Styling
* `script.js` → Frontend logic
* `server.js` → Backend server
* `package.json` → Dependencies

---

## ▶️ How to Run Locally

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start server:

   ```bash
   node server.js
   ```
4. Open `index.html` in browser
